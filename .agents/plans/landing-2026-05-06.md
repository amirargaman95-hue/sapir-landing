# Sapir Azulay — Landing + Dashboard Preview

**Project:** Sapir
**Status:** draft (awaiting GATE 2 approval)
**Created:** 2026-05-06
**Design direction:** A — Editorial Documentary (Linear / Granola / Justin Welsh / New Yorker DNA)

## What we're building

שני נכסים ב-repo אחד `sapir-landing/`:

1. **דף נחיתה ציבורי (`/`)** — מגנט לידים B2B לבעלי מפעלים בישראל. CTA יחיד: וואטסאפ → ספיר. רמת design = showcase 2026 (תיק עבודות + מנוע הפניות לאמיר).
2. **תצוגה מקדימה של פאנל אדמין (`/dashboard-preview`)** — mock UI מלא, בלי DB / auth / persistence. 5 מסכים שמציגים איך ייראה ה-CRM של ספיר.

## Why

ספיר תקבל דף נחיתה אמיתי + תמונה של מה יבוא בשלב 2 (האדמין). אמיר מקבל תיק עבודות שמציג גם UI/UX/landing וגם product UI מלא — מבדל אותו מ-99% מבוני דפי נחיתה. ספיר תמליץ → לקוחות חדשים.

## Tech stack

- **Framework:** Next.js 15 (App Router) + React 19 + Tailwind 4
- **Animation:** GSAP ScrollTrigger + framer-motion (subtle, רק ב-2-3 רגעים)
- **Icons:** Phosphor duotone
- **Typography:** Frank Ruhl Libre (כותרות, serif עברי) + Assistant (body)
- **Hosting:** Vercel
- **Color palette:** קרם חמים `#FAF8F4`, פחם `#1C1C1E`, accent כחול-טורקיז עמוק `#1F4E5F` (אקסנט אחד)

## Sections — Landing (`/`)

1. **Hero** — headline + subheadline + 2 trust signals + CTA (WhatsApp) + FUD micro-row + תמונה אמיתית של ספיר
2. **Pain agitation** — 3 כאבים של בעל מפעל (סוכנות שכשלה, עובד שעזב, זמן מבוזבז)
3. **Solution intro** — "כך אני עובדת" (3 שלבים)
4. **Video testimonials wall** — 6 סרטונים בגריד עריכותי, lightbox מלא-מסך, פוסטרים אמיתיים
5. **3 USPs** — תקופת התחייבות · אדם לא חברה · הוכחה ויזואלית (כרטיסים גדולים + Phosphor icons)
6. **How it works** — 3 צעדים ממוספרים, אסטטיקה דוקומנטרית
7. **WhatsApp evidence wall** — 9 צילומים כיצירות מוזיאון (גריד אסימטרי, ניתן להגדלה)
8. **About Sapir** — סיפור בגוף ראשון נשי + תמונה גדולה + ציטוט לריסה צף
9. **Trust badges** — שמות לקוחות (טקסט בלבד, אין לוגואים): אדמו גרופ, יוסי הנה״ח, אוהד גיוס, מאיה גיוס, שלומי
10. **FAQ** — 5 שאלות הכי נפוצות, accordion
11. **Final CTA** — כפתור וואטסאפ ענק + טופס מינימליסטי (שם · טלפון · תפקיד) שמסיים בוואטסאפ
12. **Footer** — צרו קשר + נגישות

## Sections — Dashboard Preview (`/dashboard-preview`)

1. **Login mock** — קליק אחד נכנס (אין auth)
2. **Dashboard** — 4 KPIs (השמות פעילות, מועמדים בפייפליין, לידים חדשים, לקוחות פעילים) + טיימליין השמות אחרונות
3. **Clients (לקוחות עסקיים)** — טבלת קלפים: אדמו גרופ, יוסי הנה״ח, אוהד גיוס וכו' עם היסטוריית השמות
4. **Candidates pipeline** — Kanban-style: חדש / בשיחה / נשלח למעסיק / הושמה / תקופת התחייבות
5. **Leads inbox** — לידים מהדף הציבורי (mock entries) עם action buttons
6. **Calendar mini** — לוח חודשי עם פגישות mock

כל המסכים = mock data בקובץ אחד (`_data/mock.ts`). נוויגציה sidebar.
