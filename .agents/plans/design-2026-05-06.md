## 1. Above-the-fold spec (6-element stack)

### Element 1 — Headline
> **המפעלים שגייסו דרכי — הצוות שלהם עדיין שם.**

- "So that" test: מגייסת → עובדים שנשארים → לא תתחיל מחדש → תפסיק לבזבז כסף → **הצוות שלהם עדיין שם** (הרגש)
- 8 מילים, קונקרטי, outcome-driven, חד
- Frank Ruhl Libre 800 weight, גודל clamp(2.5rem, 6vw, 5rem)
- שורתיים — נשבר אחרי "דרכי —"

### Element 2 — Subheadline
> אני ספיר. מגייסת אישית למפעלים ועסקים בישראל.
> כל השמה — תקופת התחייבות בכתב.

- 2 שורות. ראשונה = מי, שנייה = ה-USP החזק
- Assistant 400, גודל 1.125rem-1.25rem
- צבע פחם מעט מואר (`#3A3A3C`)

### Element 3 — Social proof (לא מעיד, רק מאזכר)
שורה דקה אחת תחת ה-subheadline:
> *גייסה למפעלים בישראל ב-3+ שנים האחרונות:* אדמו גרופ · יוסי הנה״ח · אוהד גיוס · מאיה גיוס · שלומי

- אין מספרים מומצאים. רק שמות (פוזיציית-trust)
- Assistant 400, גודל 0.875rem, צבע אפור (`#5A5A5C`)
- Mobile: scroll horizontally with subtle fade

### Element 4 — CTA Button
> **דברי איתי בוואטסאפ ←**

- רקע ציאן עמוק `#1F4E5F`, טקסט קרם `#FAF8F4`
- Hover: fade ל-`#163A47` + arrow shift 4px imine
- Mobile: full-width, 56px גובה (גדול יותר מ-50 = thumb-friendly)
- Desktop: inline-block, padding 18px 36px

### Element 5 — FUD micro-row (תחת ה-CTA)
> ✓ בלי טפסים   ✓ בלי סוכנויות   ✓ ספיר עונה אישית

- 3 micro-trust signals
- Phosphor `Check` icon duotone, ציאן
- Mobile: horizontal scroll קל

### Element 6 — Hero image
- **תמונה אמיתית של ספיר** (`/img/sapir-real.webp`) — פורטרט גדול
- Editorial framing: התמונה נמצאת ב-40% הימני של hero (RTL = ימני). 60% השמאלי = הטקסט (רברסי באנגלית, אבל הקנבס שלנו עברי)
- אפקט: מעט כתם נקי מ-`#E8E2D5` מתחת לתמונה כ-shadow רך, לא drop-shadow קלאסי
- אין overlay טקסט על הפנים שלה
